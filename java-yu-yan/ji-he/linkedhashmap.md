# **LinkedHashMap**

LinkedHashMap 是HashMap的一个子类（并未比HashMap实现更多的接口）。

**特性**：保存键值对的插入顺序，遍历时，先得到的键值对肯定是先插入的。

**使用场景**：需要put的顺序和遍历的顺序相同

## 数据结构

LinkedHashMap是Hash表和链表的实现，并且依靠着**双向链表**保证了迭代顺序是插入的顺序。
